# Arena-de-D-fis-Algorithmiques
Ce projet est une plateforme de défis de programmation développée en langage C.
L’objectif est d’offrir aux étudiants un ensemble de mini-jeux algorithmiques permettant :
   #d’apprendre en s’amusant
   #de développer la pensée algorithmique
   #d’améliorer leurs compétences en programmation C
   #de comparer leurs performances avec un système de score et de classement
Le projet contient plusieurs défis indépendants, tous intégrés dans une interface principale.
